## Общая информация

| Параметр | Значение |
|----------|----------|
| **Base URL** | `https://qa-internship.avito.com` |
| **API версии** | `/api/1` (основная), `/api/2` (дополнительная) |
| **Content-Type** | `application/json` |
| **Accept** | `application/json` |
| **Формат ответов** | JSON |
| **Диапазон sellerId** | `111111–999999` (рекомендованный) |

### Цель тестирования
Проверка корректности работы 4 эндпоинтов микросервиса объявлений:
- Создание объявления
- Получение объявления по ID
- Получение всех объявлений продавца
- Получение статистики по объявлению

### Структура данных

**Запрос на создание (POST /api/1/item):**
```json
{
  "sellerID": 123456,
  "name": "Название товара",
  "price": 1000,
  "statistics": {
    "likes": 0,
    "viewCount": 0,
    "contacts": 0
  }
}
```
